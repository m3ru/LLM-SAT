### Example Parameter Changes
Good changes (parameters only):
- `100.0` → `98.5` (changing a constant)
- `margin * slow` → `margin * 1.05 * slow` (adding multiplier)

Bad changes (avoid these):
- Renaming variables
- Adding new if statements
- Changing function calls
- Modifying assert statements
